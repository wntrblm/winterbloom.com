Sol

A uniquely customizable USB MIDI to CV/Gate converter powered by CircuitPython

Sol is the first Eurorack module that can be re-programmed using CircuitPython. It's designed to be useful right out of the box but flexible enough to accommodate a ton of different CV/Gate generation needs. We designed it from the start to make it easy for you to customize it.
